# Inventory Management CLI

A simple Command-Line Inventory Management System built with Python.

This project was created as part of my Python learning journey. It demonstrates the use of Python fundamentals by building a real-world application instead of solving isolated exercises.

## Features

- Add new parts
- View all inventory
- Search by part number
- Update existing parts
- Delete parts
- Save inventory to a JSON file
- Load inventory automatically when the program starts
- Basic error handling for invalid user input

## Technologies

- Python 3
- JSON
- File Handling
- Exception Handling

## Project Structure

## How to Run

1. Clone the repository:

```bash
git clone git@github.com:siavash-zr/Inventory-Management.git
```

2. Enter the project folder:

```bash
cd Inventory-Management
```

3. Run the program:

```bash
python main.py
```
